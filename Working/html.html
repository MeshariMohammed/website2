# HTML tag Description
1 !DOCTYPE html This document is HTML5 document, It must only appear once, at the top of the page
2 head This root element or beginning of HTML code that is divided into two parts (1) head and (2) body. The visible or display part is body
3 meta metainformation about the page. Meta means data about data, that means it will include all the information about and required by the page
4 title there are two usage of title tag (1) in head it is conisder as title of the page shown in chrome or IE tab, (2) in the body it is shown as tool tip text then you define it as an attribute to tag
5 body Visibile container of the HTML page
6 h1, h2, h3, h4, h5, h6 These are heading for the page where h1 is maxium and h6 is smallest. There are 6 headings, after closing of tag new line starts
7 br the tag used for \n or new line
8 /br is same as br
9 b bold the text it is used inside some tags, after the tag ends there is no new line
10 i italic the text it is used inside some tags, after the tag ends there is no new line
